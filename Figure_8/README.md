# Figure 8 – Spatial Validation of Alveolar Deficiency in BPD

This folder contains analyses and visualizations used to generate Figure 8.

## Figure 8A – Histological Assessment
Representative histological images from PT-NB, FT-NB, and BPD lung tissues showing differences in alveolar architecture.

## Figure 8B – Morphometric Analysis
Mean alveolar wall distances were quantified using ImageJ/Fiji from multiple ROIs across at least three independent donors per group. Statistical analyses and visualization were performed using GraphPad Prism.

Software:
- ImageJ/Fiji
- GraphPad Prism

## Figure 8C – Cell-Type Composition Analysis
Cell-type proportions associated with selected marker genes were calculated from the integrated Harmony-corrected and Azimuth-annotated pediatric lung snRNA-seq dataset.

Code:
- Figure8C_CellComposition.Rmd

Software:
- R
- Seurat
- ggplot2

## Figure 8D – Spatial Localization of GLB1-Associated Niches
Representative Visium HD images showing spatial localization of GLB1, RAGE, and Pan-Cytokeratin expression in BPD lung tissue.

Software:
- Space Ranger
- Loupe Browser

## Figure 8E – Spatial Gene Expression Visualization
Spatial feature maps showing SFTPC, RAGE, and GLB1 expression patterns in FT-NB and BPD lung samples.

Software:
- Space Ranger
- Loupe Browser
- Seurat

## Figure 8F – Visium HD Workflow
Schematic overview of the Visium HD spatial transcriptomics workflow.

## Figure 8G – Spatial Cell Population Summary
Comparison of epithelial, immune, endothelial, and stromal cell populations identified from Visium HD datasets.

Software:
- R
- Seurat
- ggplot2

## Figure 8H – Proposed Model
Working model summarizing altered epithelial differentiation, senescence-associated signaling, and impaired alveolar development in BPD.

## Analysis Workflow
- Histological assessment of pediatric lung tissue.
- Morphometric measurements using ImageJ/Fiji.
- Integrated snRNA-seq analysis using Harmony and Azimuth annotations.
- Visium HD spatial transcriptomic analysis using Space Ranger and Loupe Browser.
- Statistical analyses and visualization using GraphPad Prism and R.

## Software
- R
- Seurat
- ggplot2
- ImageJ/Fiji
- GraphPad Prism
- Space Ranger
- Loupe Browser
