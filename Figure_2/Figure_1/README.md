# Figure 1. Overview of Human Lung Development and Integrated snRNA-seq Analysis
This folder contains scripts used to generate Figure 1.
## Figure Components
### Panel A
Study design and analytical workflow.
### Panel B
Representative H&E images across developmental stages and BPD samples.
### Panel C
Integrated UMAP visualization of the pediatric lung snRNA-seq dataset.
### Panel D
Cell type composition analysis across developmental stages and BPD samples.
### Panel E
Epithelial pseudotime trajectory analysis.
### Panel F
Stromal pseudotime trajectory analysis.
## Input Data
Analyses were performed using the integrated pediatric lung snRNA-seq Seurat object generated during preprocessing.
## Scripts Included
- UMAP visualization
- Cell proportion analysis
- Pseudotime trajectory analysis
## Output
Scripts in this folder generate the panels corresponding to Figure 1.
