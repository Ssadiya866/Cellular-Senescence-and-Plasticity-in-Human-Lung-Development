# Mapping Cellular Senescence, Cellular Plasticity, and Alveolar Niche Heterogeneity in Human Lung Development
This repository contains the code, workflows, and source files used for the analyses presented in the manuscript.
## Contents
- Preprocessing and integration of pediatric lung snRNA-seq datasets
- Harmony integration and Azimuth annotation workflows
- Cell proportion analyses
- Differential gene expression analyses
- Gene ontology and pathway enrichment analyses
- Heatmap generation
- Feature plots and visualization scripts
- Pseudotime trajectory analyses
- Molecular clock and senescence-related analyses
The repository is organized by figure to facilitate reproducibility and code availability.
